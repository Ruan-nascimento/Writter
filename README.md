# ✍️ **Writter**

A simple web app to share and discover quotes and thoughts. Developed using **JavaScript**, with **React** for the interface and **TailwindCSS** for practical and responsive design. This project was created with the goal of testing and improving web development skills. 🚀

## 📝 **Project Description**

**Writter** is a platform for posting short texts, functioning like a social network for writers of quotes and thoughts. The main idea is to:

- 📌 Create posts and share them with other users
- 🌟 Give a star to the posts you like
- 💬 Leave comments on posts
- 👤 Customize your profile with a photo and personal information
- 📝 Save your favorite posts
- 🔍 Search for specific posts
- 👀 See other "writers" who are online

## 🚀 **Technologies Used**

- **JavaScript**: The main language of the project
- **React**: Framework for building the interface
- **TailwindCSS**: Framework for practical and responsive styling

## ⚙️ **Features**

- **Sign Up and Login**: Register or log in to access the app
- **Posts**: Write and view posts, like, comment, and give a star if you like the post
- **Profile Settings**: Customize your photo and personal information
- **Save Posts**: Save your favorite posts to access them later
- **View "Writers" Online**: Check who is online
- **Search**: Look for specific posts of interest
